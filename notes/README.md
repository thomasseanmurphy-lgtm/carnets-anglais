# notes/ — idées futures (non construites)

Ce dossier garde les **idées décidées « on y pense »**, pas encore construites. Une idée = **un
fichier**. On y met le *besoin*, le *pourquoi*, les *contraintes connues* et des *pistes* — **pas
de code**.

Ne pas confondre :

| Dossier / fichier | Contient | Question à laquelle il répond |
|---|---|---|
| [`../docs/`](../docs/) | ce qui **existe aujourd'hui** | « comment ça marche / se reproduit ? » |
| [`../CHANGELOG.md`](../CHANGELOG.md) | ce qui **a été fait** | « qu'est-ce qui a changé, quand ? » |
| **`notes/`** (ici) | ce qu'on **veut faire un jour** | « quelles idées garder ? » |

Quand une idée passe en construction, elle quitte `notes/` : le fonctionnement final va dans
`docs/`, et l'ajout est consigné dans `CHANGELOG.md`.
